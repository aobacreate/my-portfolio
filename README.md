
私のポートフォリオをnext.jsで作りました。

## Demo

▶ https://my-portfolio-one-sandy-71.vercel.app

## 構成

- **Heroページ**
  - Framer Motionでタイトルをアニメーション表示
  - Animation Frameを使ってCubeを回転

- **Worksページ**
  - 制作した作品をリスト表示
  - Framer Motionのlayout機能で開閉UIを実装

- **Skillsページ**
  - 最近使用しているスキルを表示

- **Footer**
  
## 使用技術

Next.js / Framer Motion / TypeScript / Tailwind CSS 

## セットアップ

以下のコマンドで開発サーバーを起動できます。

サーバー上で
```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000)でブラウザーからアクセスできます。
